# 아ㅏ 모야~~~~~~~~~~~.....

Next.js 기반 웹 애플리케이션

## 시작하기

### 의존성 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

개발 서버는 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm start
```

## 기술 스택

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Radix UI

## 프로젝트 구조

- `/src/app` - Next.js App Router 페이지
- `/src/components` - React 컴포넌트
- `/src/commons` - 공통 컴포넌트 및 유틸리티
